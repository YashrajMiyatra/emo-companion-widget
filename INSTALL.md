# EMO Desktop Widget - Installation Guide

Welcome! If you've just downloaded this project and want to get the robot running on your screen, follow these simple steps.

## Prerequisites
Before you begin, you **must** have Node.js installed on your computer. 
1. Go to [nodejs.org](https://nodejs.org/).
2. Download and install the "LTS" (Long Term Support) version for Windows.
3. Keep clicking "Next" through the installer with the default settings until it finishes.

---

## Easiest Installation (Windows Only)

1. Extract the downloaded folder to your Desktop or somewhere safe.
2. Inside the folder, find the file named **`install.bat`**. 
3. **Double-click `install.bat`**. A black window will pop up. It will take a minute to download the required files (like Electron) from the internet. When it's done, it will say "Press any key to continue...". You can close that window.
4. You only ever have to do Step 3 once!

## How to Start the Robot

1. Whenever you want to launch your companion, just double-click the **`start.bat`** file in the folder!
2. The robot will appear on the bottom-right of your screen.

## How to Stop the Robot
Just **Right-Click** anywhere on the robot's dark face, and it will close itself instantly.

---

## For Developers (Mac/Linux/Windows Terminal)
If you prefer using the command line:
1. Open your terminal.
2. `cd` into the downloaded folder directory.
3. Run `npm install`
4. Run `npm start`
